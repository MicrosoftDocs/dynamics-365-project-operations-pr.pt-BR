# <a name="actuals-impact-in-a-time-and-materials-engagement"></a>Impacto dos dados reais em um compromisso de tempo e material

_**Aplica-se a:** Project Operations para cenários baseados em recursos/não estocados, implantação Lite - transação para faturamento pro forma_

A tabela a seguir lista os dados reais de diferentes tipos de transação criados em vários eventos em um compromisso de tempo e material.

| Evento | Custo real | Valores reais de vendas não cobrados | Dados reais de vendas cobradas | Exemplo |
|---|---|---|---|---|
| As horas são criadas. | Não aplicável | Não aplicável | Não aplicável | <p>Bob Kozack, da unidade organizacional da Fabrikam US que tem uma taxa de custo de 100 dólares americanos (USD 100) por hora, está trabalhando em um projeto chamado "Instalação de braço na Adatum". Para este projeto, a taxa de cobrança contratada é de USD 200 por hora. Este é um exemplo de entrada de hora de Bob Kozak:</p><p>Bob Kozack, 8 horas</p> |
| As horas são enviadas. | Não aplicável | Não aplicável | Não aplicável | Uma linha do diário de custo e diários de vendas não cobradas são criados para a entrada de hora. O preço padrão e a taxa de custo são inseridos na entrada de diário. |
| A entrada de hora é recuperada antes de ser aprovada. | Não aplicável | Não aplicável | Não aplicável | |
| As horas são aprovadas: as horas enviadas são iguais às horas faturáveis. | Os dados reais de custo são criados. | Os dados reais de vendas não cobradas são criados. | Não aplicável | <p>Os novos dados reais que são criados:</p><ul><li>**Dados reais de custo:** Bob Kozack, 8 h, USD 800</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, 8 h, USD 1.600</li></ul> |
| As horas são aprovadas: as horas faturáveis são reduzidas e ficam abaixo das horas enviadas. | Os dados reais de custo são criados. | <p>Os dados reais de duas vendas não cobradas são criados:</p><ul><li>Dados reais de vendas não cobradas passíveis de cobrança referentes às horas faturáveis</li><li>Dados reais de vendas não cobradas não passíveis de cobrança referentes à quantidade restante</li></ul> | Não aplicável | <p>Os novos dados reais que são criados:</p><ul><li>**Dados reais de custo:** Bob Kozack, 8 h, USD 800</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, 6 h, USD 1.200, *Passível de Cobrança*</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, 2 h, USD 400, *Não Passível de Cobrança*</li></ul> |
| As horas são aprovadas: as horas faturáveis excederam as horas enviadas. | Os dados reais de custo são criados. | Os dados reais de vendas não cobradas são criados. | Não aplicável | <p>Os novos dados reais que são criados:</p><ul><li>**Dados reais de custo:** Bob Kozack, 8 h, USD 800</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, 10 h, USD 2.000</li></ul> |
| A aprovação de horas é cancelada. | <p>O status de ajuste dos dados de custo real originais é atualizado para **Ajustado**.</p><p>Dados reais de custo de reversão são criados com o status de ajuste **Não Ajustável**.</p> | <p>O status de ajuste dos dados reais originais de vendas não cobradas é atualizado para **Ajustado**.</p><p>Os dados reais de uma venda não cobrada com reversão são criados com o status de ajuste **Não Ajustável**.</p> | Não aplicável | <p>Os dados reais existentes que são atualizados:</p><ul><li>**Custo real:** Bob Kozack, 8 h, USD 800, *Ajustado*</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, 8 h, USD 1.600, *Ajustado*</li></ul><p>Os novos dados reais que são criados para reverter o impacto financeiro anterior:</p><ul><li>**Dados reais de custo:** Bob Kozack, (8 h), (USD 800), *Não Ajustável*</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, (8 h), (USD 1.600), *Não Ajustável*</li></ul> |
| A entrada de hora é recuperada depois de ser aprovada. | <p>O status de ajuste dos dados de custo real originais é atualizado para **Ajustado**.</p><p>Dados reais de custo de reversão são criados com o status de ajuste **Não Ajustável**.</p> | <p>O status de ajuste dos dados reais originais de vendas não cobradas é atualizado para **Ajustado**.</p><p>Os dados reais de uma venda não cobrada com reversão são criados com o status de ajuste **Não Ajustável**.</p> | Não aplicável | <p>Os dados reais existentes que são atualizados:</p><ul><li>**Custo real:** Bob Kozack, 8 h, USD 800, *Ajustado*</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, 8 h, USD 1.600, *Ajustado*</li></ul><p>Os novos dados reais que são criados para reverter o impacto financeiro anterior:</p><ul><li>**Dados reais de custo:** Bob Kozack, (8 h), (USD 800), *Não Ajustável*</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, (8 h), (USD 1.600), *Não Ajustável*</li></ul> |
| O contrato é confirmado. | <p>O status de ajuste dos antigos dados reais de custo foi atualizado para **Ajustado**.</p><p>Dados reais de custo de reversão são criados com o status de ajuste **Não Ajustável**.</p><p>Novos dados reais de custo são criados depois que as regras contratuais são reavaliadas.</p> | <p>O status de ajuste dos dados reais antigos de vendas não cobradas é atualizado para **Ajustado**.</p><p>Dados reais de vendas não cobradas com reversão são criados com o status de ajuste **Não Ajustável**.</p><p>Novos dados reais de vendas não cobradas são criados depois que as regras contratuais são reavaliadas.</p> | Não aplicável | <p>Os dados reais existentes que são atualizados:</p><ul><li>**Custo real:** Bob Kozack, 8 h, USD 800, *Ajustado*</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, 8 h, USD 1.600, *Ajustado*</li></ul><p>Os novos dados reais que são criados para reverter o impacto financeiro anterior:</p><ul><li>**Dados reais de custo:** Bob Kozack, (8 h), (USD 800), *Não Ajustável*</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, (8 h), (USD 1.600), *Não Ajustável*</li></ul><p>Os novos dados reais que são criados relacionados ao impacto financeiro reavaliado:</p><ul><li>**Dados reais de custo:** Bob Kozack, 8 h, USD 800</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, 8 h, USD 1.600</li></ul> |
| Uma fatura é criada. | Não aplicável | Não aplicável | Não aplicável | |
| A fatura é confirmada. Não há alteração de quantidade nos detalhes da linha de fatura em relação à quantidade nos dados reais de vendas não cobradas. | Não aplicável | <p>O status da fatura dos dados reais antigos de vendas não cobradas é atualizado.</p><p>Dados reais de vendas não cobradas com reversão são criados com o status de ajuste **Não Ajustável**. | Os dados reais de vendas cobradas são criados. | <p>Dados reais existentes que permanecem inalterados:</p><ul><li>**Dados reais de custo:** Bob Kozack, 8 h, USD 800</li></ul><p>Os dados reais existentes que são atualizados:</p><ul><li>**Dados reais de vendas não cobradas:** Bob Kozack, 8 h, USD 1.600, *Fatura do Cliente Lançada*</li></ul>Os novos dados reais que são criados para reverter o WIP (trabalho em andamento) financeiro:</p><ul><li>**Dados reais de vendas não cobradas:** Bob Kozack, (8 h), (USD 1.600)</li></ul><p>Novos dados reais que são criados para registrar os valores de vendas cobradas:</p><ul><li>**Dados reais de vendas cobradas:** Bob Kozack, 8 h, USD 1.600</li></ul> |
| A fatura é confirmada depois que a quantidade nos detalhes da linha de fatura é diminuída em relação à quantidade nos dados reais de vendas não cobradas. | Não aplicável | <p>O status de ajuste dos dados reais originais de vendas não cobradas é atualizado para **Ajustado**.</p><p>Dados reais de vendas não cobradas com reversão são criados para os dados reais originais de vendas não cobradas. Eles têm o status de ajuste **Não Ajustável**.</p><p>Os dados reais de duas novas vendas não cobradas são criados:</p><ul><li>Dados reais de vendas não cobradas passíveis de cobrança referentes às horas faturáveis</li><li>Dados reais de vendas não cobradas não passíveis de cobrança referentes à quantidade restante</li></ul><p>Dados reais de vendas não cobradas com reversão são criados para os dados reais de duas novas vendas não cobradas.</p> | <p>Os dados reais de duas vendas cobradas são criados:</p><ul><li>Dados reais de vendas cobradas passíveis de cobrança referentes às horas faturáveis</li><li>Dados reais de vendas cobradas não passíveis de cobrança referentes à quantidade restante</li></ul> | <p>Dados reais existentes que permanecem inalterados:</p><ul><li>**Dados reais de custo:** Bob Kozack, 8 h, USD 800</li></ul><p>Os dados reais existentes que são atualizados:</p><ul><li>**Dados reais de vendas não cobradas:** Bob Kozack, 8 h, USD 1.600, *Ajustado*</li></ul><p>Novos dados reais que são criados para reverter o WIP financeiro anterior:</p><ul><li>**Dados reais de vendas não cobradas:** Bob Kozack, (8 h), (USD 1.600), *Não Ajustável*</li></ul><p>Novos dados reais que são criados para registrar o WIP de vendas atualizado:</p><ul><li>**Dados reais de vendas não cobradas:** Bob Kozack, 6 h, USD 1.200, *Passível de Cobrança*</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, 2 h, USD 400, *Não Passível de Cobrança*</li></ul><p>Novos dados reais que são criados para reverter o WIP de vendas atualizado:</p><ul><li>**Dados reais de vendas não cobradas:** Bob Kozack, (6 h), (USD 1.200), *Passível de Cobrança*</li><li>**Dados reais de vendas não cobradas:** Bob Kozack, (2 h), (USD 400), *Não
