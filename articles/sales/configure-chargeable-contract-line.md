# <a name="configure-chargeable-components-of-a-project-based-contract-line"></a><span data-ttu-id="f9e5d-103">Configurar componentes passíveis de cobrança de uma linha de contrato com base em projeto</span><span class="sxs-lookup"><span data-stu-id="f9e5d-103">Configure chargeable components of a project-based contract line</span></span>

<span data-ttu-id="f9e5d-104">_**Aplicável A:** Project Operations para cenários baseados em recursos/sem estoque_</span><span class="sxs-lookup"><span data-stu-id="f9e5d-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios_</span></span>

<span data-ttu-id="f9e5d-105">Uma linha de contrato com base em projeto tem o conceito de componentes incluídos, passíveis de cobrança e não passíveis de cobrança.</span><span class="sxs-lookup"><span data-stu-id="f9e5d-105">A project-based contract line has the concept of included, chargeable, and non-chargeable components.</span></span>

<span data-ttu-id="f9e5d-106">Os componentes incluídos estão sujeitos ao método de cobrança, divisões do cliente, limites que não devem ser excedidos e configurações de frequência de fatura definidas na linha do contrato com base no projeto.</span><span class="sxs-lookup"><span data-stu-id="f9e5d-106">Included components are subject to the billing method, customer splits, not-to-exceed limits, and invoice frequency settings defined on the project-based contract line.</span></span>

<span data-ttu-id="f9e5d-107">Um subconjunto dos componentes incluídos pode ser marcado como passível de cobrança atualizando o campo **Tipo de Cobrança**.</span><span class="sxs-lookup"><span data-stu-id="f9e5d-107">A subset of the included components can be marked as chargeable by updating the **Billing Type** field.</span></span>

<span data-ttu-id="f9e5d-108">Os componentes passíveis de cobrança podem ser definidos em funções e categorias de transação.</span><span class="sxs-lookup"><span data-stu-id="f9e5d-108">Chargeable components can be defined on roles, and transaction categories.</span></span>

<span data-ttu-id="f9e5d-109">Para uma linha de contrato de projeto, os encargos definidos em uma função só se aplica à classe de transação **Tempo**.</span><span class="sxs-lookup"><span data-stu-id="f9e5d-109">For a project contract line, the chargeability defined on a role only applies to the **Time** transaction class.</span></span> <span data-ttu-id="f9e5d-110">Se **Incluir Tempo** estiver configurado como **Não** em uma linha de contrato de projeto, a guia **Funções passíveis de cobrança** não estará disponível.</span><span class="sxs-lookup"><span data-stu-id="f9e5d-110">If **Include Time** is set to **No** on a project contract line, the **Chargeable roles** tab isn't available.</span></span>

<span data-ttu-id="f9e5d-111">Os encargos são definidos em categorias de transação para uma linha de contrato do projeto e só se aplicam à classe de transação **Despesa**.</span><span class="sxs-lookup"><span data-stu-id="f9e5d-111">Chargeability defined on transaction categories for a project contract line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="f9e5d-112">Se **Incluir Despesas** estiver configurado como **Não** em uma linha de contrato de projeto, a guia **Categorias Passíveis de Cobrança** não estará disponível.</span><span class="sxs-lookup"><span data-stu-id="f9e5d-112">If **Include Expenses** is set to **No** on a project contract line, the **Chargeable Categories** tab isn't available.</span></span>

<span data-ttu-id="f9e5d-115">Na guia **Funções passíveis de cobrança** de uma linha de contrato com base em projeto, na subgrade **Categorias Passíveis de Cobrança** , no campo **Tipo de Cobrança** , atualize o tipo de cobrança para uma função.</span><span class="sxs-lookup"><span data-stu-id="f9e5d-115">On the **Chargeable roles** tab of a projec-based contract line, on the **Chargeable Categories** sub-grid, in the **Billing Type** field, update the billing type for a role.</span></span>

### <a name="update-a-transaction-category-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="f9e5d-116">Atualizar uma categoria de transação para passível de cobrança ou não passível de cobrança</span><span class="sxs-lookup"><span data-stu-id="f9e5d-116">Update a transaction category to be chargeable or non-chargeable</span></span>

<span data-ttu-id="f9e5d-117">Uma categoria de transação pode ser passível de cobrança ou não passível de cobrança em uma linha de contrato com base em projeto específica.</span><span class="sxs-lookup"><span data-stu-id="f9e5d-117">A transaction category can be chargeable or non-chargeable on a specific project-based contract line.</span></span>

<span data-ttu-id="f9e5d-118">Na guia **Categorias Passíveis de Cobrança** de uma linha de contrato com base em projeto, na subgrade **Categorias Passíveis de Cobrança** , no campo **Tipo de Cobrança** , atualize o tipo de cobrança para uma transação.</span><span class="sxs-lookup"><span data-stu-id="f9e5d-118">On the **Chargeable Categories** tab of a project-based contract line, on the **Chargeable Categories** sub-grid, in the **Billing Type** field, update the billing type for a transaction.</span></span>
