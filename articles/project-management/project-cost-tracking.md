## <a name="labor-cost-tracking-view"></a><span data-ttu-id="a42d8-108">Exibição Acompanhamento de custos de mão de obra</span><span class="sxs-lookup"><span data-stu-id="a42d8-108">Labor Cost Tracking view</span></span>

<span data-ttu-id="a42d8-109">Na página **Projetos**, na guia **Acompanhamento**, você pode selecionar **Acompanhamento** > **Custo** para abrir a exibição **Acompanhamento de Custos** e ver o andamento da mão de obra gasta em cada tarefa no plano do projeto.</span><span class="sxs-lookup"><span data-stu-id="a42d8-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="a42d8-110">Essa exibição também compara o custo de mão de obra real gasto em uma tarefa com o custo de mão de obra planejado da tarefa.</span><span class="sxs-lookup"><span data-stu-id="a42d8-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="a42d8-111">O Project Operations usa as seguintes fórmulas para calcular métricas de custo de mão de obra:</span><span class="sxs-lookup"><span data-stu-id="a42d8-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>

- <span data-ttu-id="a42d8-112">**Custo planejado**: Custo estimado de todas as atribuições de recursos em cada tarefa do nó folha</span><span class="sxs-lookup"><span data-stu-id="a42d8-112">**Planned cost**: Estimated cost of all resource assignments on each leaf node task</span></span>

<span data-ttu-id="a42d8-118">Cada tarefa mostra uma projeção da variação de custo na tarefa.</span><span class="sxs-lookup"><span data-stu-id="a42d8-118">Each task shows a projection of the cost variance on the task.</span></span> <span data-ttu-id="a42d8-119">Se a estimativa de custo completa for maior do que o custo planejado, a tarefa será projetada para ultrapassar o orçamento.</span><span class="sxs-lookup"><span data-stu-id="a42d8-119">If the cost estimate at complete is more than the planned cost, the task is projected to go over budget.</span></span> <span data-ttu-id="a42d8-120">Se a estimativa de custo completa for menor do que o custo planejado, a tarefa será projetada para terminar dentro do orçamento.</span><span class="sxs-lookup"><span data-stu-id="a42d8-120">If the cost estimate at complete is less than the planned cost, the task is projected to finish under budget.</span></span>

<span data-ttu-id="a42d8-122">Todos os valores de custo mostrados são convertidos na moeda de custo do projeto a partir da moeda do preço de custo do projeto usada para determinar a taxa de custo.</span><span class="sxs-lookup"><span data-stu-id="a42d8-122">All cost amounts shown are converted to the project's cost currency from the project's cost price currency used to determine the cost rate.</span></span> <span data-ttu-id="a42d8-123">A moeda do custo do projeto é a moeda da unidade contratante no projeto.</span><span class="sxs-lookup"><span data-stu-id="a42d8-123">The cost currency of the project is the currency of the contracting unit on the project.</span></span> <span data-ttu-id="a42d8-124">Os valores de custo estimado para tempo que são mostrados na guia **Estimativas** na página **Projeto** podem não ser adicionados ao custo planejado na guia **Acompanhamento**. >
> - <span data-ttu-id="a42d8-125">A guia **Estimativas** calcula o custo estimado usando a mesma moeda da taxa de custo na lista de preços.</span><span class="sxs-lookup"><span data-stu-id="a42d8-125">The **Estimates tab** calculates the estimated cost by using the same currency of the cost rate in the price list.</span></span> <span data-ttu-id="a42d8-126">O custo estimado na moeda da lista de preços é convertido no custo estimado na moeda de custo do projeto.</span><span class="sxs-lookup"><span data-stu-id="a42d8-126">Then, the estimated cost in the price list currency is converted to the estimated cost in the project's cost currency.</span></span> <span data-ttu-id="a42d8-127">O custo estimado na moeda do projeto é arredondado para 2 casas decimais.</span><span class="sxs-lookup"><span data-stu-id="a42d8-127">The estimated cost in the project currency is shown rounded to 2 decimal places.</span></span> <span data-ttu-id="a42d8-128">Em nenhum ponto dessa conversão a precisão da moeda é aplicada.</span><span class="sxs-lookup"><span data-stu-id="a42d8-128">At no point during this conversion is currency precision applied.</span></span> 
> - <span data-ttu-id="a42d8-129">Na guia **Acompanhamento**, o cálculo do custo planejado segue uma ordem de cálculo ligeiramente diferente que envolve a aplicação da precisão da moeda em dois estágios:</span><span class="sxs-lookup"><span data-stu-id="a42d8-129">On the **Tracking** tab, the planned cost calculation follows a slightly different calculation order that involves currency precision being applied in two stages:</span></span>

   ></ol>
   ><span data-ttu-id="a42d8-132">A precisão da moeda é aplicada em ambas as etapas para obter um custo planejado (na guia **Acompanhamento**) que se desvia um pouco do custo estimado (na exibição **Tempo - em fases** na guia **Estimativas**).</span><span class="sxs-lookup"><span data-stu-id="a42d8-132">Currency precision is applied in both steps to get a planned cost (on the **Tracking** tab) that deviates slightly from the estimated cost (on the **Time - phased** view on the **Estimates** tab).</span></span>
