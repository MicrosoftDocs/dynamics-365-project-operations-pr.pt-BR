# <a name="set-up-and-use-pay-when-paid-vendor-payments"></a><span data-ttu-id="81e4b-103">Configurar e usar pagamentos de fornecedor pagar-quando-pago</span><span class="sxs-lookup"><span data-stu-id="81e4b-103">Set up and use pay-when-paid vendor payments</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="81e4b-104">Ao aprovar um fornecedor para trabalhar como subcontratado, talvez você queira reter o pagamento ao fornecedor até que o cliente pague pelo projeto.</span><span class="sxs-lookup"><span data-stu-id="81e4b-104">When you approve a vendor to work as a subcontractor, you might want to withhold payment to the vendor until your customer pays you for the project.</span></span> <span data-ttu-id="81e4b-105">Para oferecer suporte a esse cenário, você pode configurar os termos de pagar-quando-pago (PWP) ao configurar a ordem de compra (OC) com o fornecedor.</span><span class="sxs-lookup"><span data-stu-id="81e4b-105">To support this scenario, you can set up pay-when-paid (PWP) terms when you set up the purchase order (PO) with the vendor.</span></span>

## <a name="example"></a><span data-ttu-id="81e4b-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="81e4b-110">Example</span></span>

<span data-ttu-id="81e4b-111">Sua organização concorda em fornecer a um cliente 100 computadores com software instalado, por um preço de 150,00 dólares americanos (USD) por computador.</span><span class="sxs-lookup"><span data-stu-id="81e4b-111">Your organization agrees to provide a customer with 100 computers that have software installed, for a price of 150.00 US dollars (USD) per computer.</span></span> <span data-ttu-id="81e4b-112">Depois, você contrata um fornecedor para fornecer os computadores que possuem o software instalado.</span><span class="sxs-lookup"><span data-stu-id="81e4b-112">You then hire a vendor to provide you with the computers that have software installed.</span></span> <span data-ttu-id="81e4b-113">De acordo com o contrato, o cliente deve aprovar a qualidade dos computadores antes de pagar sua organização.</span><span class="sxs-lookup"><span data-stu-id="81e4b-113">According to the agreement, the customer must approve the quality of the computers before it pays your organization.</span></span> <span data-ttu-id="81e4b-114">A política da sua organização consiste em reter o pagamento aos fornecedores até que o cliente pague a você.</span><span class="sxs-lookup"><span data-stu-id="81e4b-114">Your organization's policy is to withhold payment to vendors until the customer has paid you.</span></span> <span data-ttu-id="81e4b-115">Portanto, você configura o projeto para que tenha uma porcentagem de PWP de 100%.</span><span class="sxs-lookup"><span data-stu-id="81e4b-115">Therefore, you set up the project so that it has a PWP percentage of 100 percent.</span></span>

<span data-ttu-id="81e4b-116">O fornecedor envia para você os 100 computadores que possuem o software instalado, juntamente com uma fatura de 10.000,00 USD.</span><span class="sxs-lookup"><span data-stu-id="81e4b-116">The vendor sends you the 100 computers that have software installed, together with an invoice for 10,000.00 USD.</span></span> <span data-ttu-id="81e4b-117">Como os termos de PWP são configurados para o seu projeto, você não paga o fornecedor no recebimento dos computadores.</span><span class="sxs-lookup"><span data-stu-id="81e4b-117">Because PWP terms are set up for your project, you don't pay the vendor upon receipt of the computers.</span></span> <span data-ttu-id="81e4b-118">Em vez disso, você envia os computadores ao cliente, juntamente com uma fatura de 15.000,00.</span><span class="sxs-lookup"><span data-stu-id="81e4b-118">Instead, you send the computers to the customer, together with an invoice for 15,000.00.</span></span> <span data-ttu-id="81e4b-119">O cliente inspeciona os computadores e aprova o valor total da fatura do projeto.</span><span class="sxs-lookup"><span data-stu-id="81e4b-119">The customer inspects the computers and approves the full amount of the project invoice.</span></span>
