<span data-ttu-id="5e506-105">Uma linha de cotação com base em projeto tem o conceito de componentes *incluídos* e componentes *passíveis de cobrança*.</span><span class="sxs-lookup"><span data-stu-id="5e506-105">A project-based quote line has the concept of *included* components and *chargeable* components.</span></span>

<span data-ttu-id="5e506-106">Os componentes incluídos estão sujeitos a:</span><span class="sxs-lookup"><span data-stu-id="5e506-106">Included components are subject to:</span></span>

  - <span data-ttu-id="5e506-109">Configurações de frequência de fatura definidas na linha de cotação com base em projeto</span><span class="sxs-lookup"><span data-stu-id="5e506-109">Invoice frequency settings defined on the project-based quote line</span></span>

<span data-ttu-id="5e506-110">Um subconjunto dos componentes incluídos pode ser marcado como cobrável usando o campo **Tipo de Cobrança**.</span><span class="sxs-lookup"><span data-stu-id="5e506-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="5e506-111">O campo **Tipo de Cobrança** é um conjunto de opções que permite os seguintes valores no contexto de uma linha de cotação:</span><span class="sxs-lookup"><span data-stu-id="5e506-111">The **Billing Type** field is an option-set that allows the following values in the context of a quote line:</span></span>

  - <span data-ttu-id="5e506-113">Não Passível de Cobrança</span><span class="sxs-lookup"><span data-stu-id="5e506-113">Non-chargeable</span></span>

<span data-ttu-id="5e506-114">Os componentes passíveis de cobrança podem ser definidos em tarefas, funções e categorias de transação.</span><span class="sxs-lookup"><span data-stu-id="5e506-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="5e506-121">Atualizar uma tarefa de projeto para passível de cobrança ou não passível de cobrança</span><span class="sxs-lookup"><span data-stu-id="5e506-121">Update a project task to be chargeable or non-chargeable</span></span>

<span data-ttu-id="5e506-122">Uma tarefa de projeto pode ser passível de cobrança ou não passível de cobrança no contexto de uma linha de cotação com base em projeto, o que torna possível a seguinte configuração:</span><span class="sxs-lookup"><span data-stu-id="5e506-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>

<span data-ttu-id="5e506-123">Se uma linha de cotação com base em projeto incluir **Tempo** e a tarefa **T1**, a tarefa será associada à linha de cotação como passível de cobrança.</span><span class="sxs-lookup"><span data-stu-id="5e506-123">If a project-based quote line includes **Time** and the task **T1**, the task is associated to the quote line as chargeable.</span></span> <span data-ttu-id="5e506-124">Se houver uma segunda linha de cotação que inclua **Despesas**, você poderá associar a tarefa **T1** na linha de cotação como não passível de cobrança.</span><span class="sxs-lookup"><span data-stu-id="5e506-124">If there is a second quote line that includes **Expenses**, you can associate the **T1** task on the quote line as non-chargeable.</span></span> <span data-ttu-id="5e506-125">O resultado é que todo o tempo registrado na tarefa é passível de cobrança e todas as despesas registradas na tarefa não são passíveis de cobrança.</span><span class="sxs-lookup"><span data-stu-id="5e506-125">The result is that all time recorded on the task is chargeable and all expenses recorded on the task are non-chargeable.</span></span>
