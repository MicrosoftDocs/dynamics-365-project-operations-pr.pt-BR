<span data-ttu-id="6733a-105">Uma linha de cotação com base em projeto tem o conceito de componentes *incluídos* e componentes *passíveis de cobrança*.</span><span class="sxs-lookup"><span data-stu-id="6733a-105">A project-based quote line has the concept of *included* components and *chargeable* components.</span></span>

<span data-ttu-id="6733a-106">Os componentes incluídos estão sujeitos a:</span><span class="sxs-lookup"><span data-stu-id="6733a-106">Included components are subject to:</span></span>

  - <span data-ttu-id="6733a-109">Configurações de frequência de fatura definidas na linha de cotação com base em projeto</span><span class="sxs-lookup"><span data-stu-id="6733a-109">Invoice frequency settings defined on the project-based quote line</span></span>

<span data-ttu-id="6733a-110">Um subconjunto dos componentes incluídos pode ser marcado como cobrável usando o campo **Tipo de Cobrança**.</span><span class="sxs-lookup"><span data-stu-id="6733a-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="6733a-111">O campo **Tipo de Cobrança** é um conjunto de opções que permite os seguintes valores no contexto de uma linha de cotação:</span><span class="sxs-lookup"><span data-stu-id="6733a-111">The **Billing Type** field is an option-set that allows the following values in the context of a quote line:</span></span>

  - <span data-ttu-id="6733a-113">Não Passível de Cobrança</span><span class="sxs-lookup"><span data-stu-id="6733a-113">Non-chargeable</span></span>

<span data-ttu-id="6733a-114">Os componentes passíveis de cobrança podem ser definidos em tarefas, funções e categorias de transação.</span><span class="sxs-lookup"><span data-stu-id="6733a-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="6733a-121">Atualizar uma tarefa de projeto para passível de cobrança ou não passível de cobrança</span><span class="sxs-lookup"><span data-stu-id="6733a-121">Update a project task to be chargeable or non-chargeable</span></span>

<span data-ttu-id="6733a-122">Uma tarefa de projeto pode ser passível de cobrança ou não passível de cobrança no contexto de uma linha de cotação com base em projeto, o que torna possível a seguinte configuração:</span><span class="sxs-lookup"><span data-stu-id="6733a-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>
